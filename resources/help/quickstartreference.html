Quick Start Reference

Some of the most frequently used commands can be found here.
Note that it's NOT the full command reference!

The commands are grouped in a number of sections, starting with the ones which don't have a graphical representation, like Project Management, and (partially) the Song.

Sections

Key actions are divided in the following categories:

  • Global commands
  • Song commands
  • Track commands
  • AudioClip commands

Global commands

< H > This Help Window ;-)
< F4 > Switch to the Project Manager View.
Here you can manage your Project, and settings related to the audio device, keyboard, and so on.
< F3 > Switch to the SongView.
Displays the currently active Song.
< Q > Show the "Context Menu" for the grahpical object the mouse cursors points to.
n.. < S > Switch to Song n
Typ in the number of the Song you want to show, and type S.
The "n.." means "Number Collection".
< CTRL S > Save the Project.
< < ESC > > Quit Traverso (Do you really want to do this? ;-) ).
Of course, you can use the red x button to exit Traverso too.
Your Project will be saved automatically!
< < ENTER > > Render Song - Project

Song commands

< SPACE > Go !
Saying it a bit differently, start playing ;-).
Issueing the same command again will stop the playing.
If a Track is "armed" it will also start recording in that Track!
< W > Move the "Play Cursor" to the "Edit Cursor", a.k.a. the "Work Cursor".
< LEFT SHIFT > Set's the "Work Cursor" to the current mouse cursor position.
< HOME > Move the "Work Cursor" to the start of the Song.
[ Z ] Omnidirectional Zooming.
< Z > Zoom in
< Z X > Zoom out
[ TAB ] The famous Shuttle Tool !!
The Shuttle tool gives you continues horizontall scroll. The speed is set by the mouse position.
< S N > Toggle Snap on - off.
< F5 > Undo
< F6 > Redo
< S > Select an AudioClip
< < C X > > Delete all AudioClips in Selection.
This action is un/redoable!
Another way of removing an AudioClip is to drag it outside of the SongView at the left or top of the view!

Track commands

< A > Arm the Track.
The next time the Song starts playing, this Track will record on the specified Capture Bus.
< E > Edit Track information (Like Track Name)
< V B > Select a Capture Bus to record from.
< O > Solo
All the other Tracks will be silenced.
[ O ] Solo this Track, until the hold actions finishes.
A proper name would perhaps "silence others" ?
< U > Mute
[ G ] Change Gain.
